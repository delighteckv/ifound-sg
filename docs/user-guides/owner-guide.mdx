# Owner User Guide

## Purpose

The Owner is a customer who receives QR code stickers from iFound and registers those QR codes to personal items such as a MacBook, bicycle, backpack, keys, or luggage.

## Primary Responsibilities

- Create and access an iFound account
- claim the pack by PackId
- Register valid QR codes received in a pack
- Link each QR code to a specific owned item
- Monitor scans, finder messages, and calls
- Manage profile, linked sign-in methods, and subscription

## Supported Platforms

- Web dashboard
- Android app
- iOS app

## Account Access

Owners can sign in using:

- Email and password
- Phone OTP
- Google
- Apple

Where supported, the same person should continue using the same account identity.

## First-Time Owner Flow

### Step 1: Sign Up

1. Open the iFound app or website.
2. Sign up using email, phone, Google, or Apple.
3. Complete verification if required.
4. Sign in to your account.

### Step 2: Register a QR Code

1. Open `QR Codes` in the dashboard or app.
2. Click or tap `Register QR Code`.
3. Enter the QR code printed on your sticker or pack.
4. Enter what the QR code is attached to.
   - Example: `MacBook Pro`
   - Example: `Bicycle`
5. Select a category.
6. Optionally add a description.
7. Submit.

### Expected Result

- The system validates that the QR code exists.
- The system validates that the QR code is not already assigned.
- The QR code is linked to your account.
- The QR code is linked to your item record.

## Registering Items

You should register one QR code per item.

Examples:

- Laptop
- Bicycle
- School bag
- Wallet
- Toy
- Keys

For each registration, the QR code confirms that you are the owner of that tagged item within the platform.

## QR Code Validation Rules

Registration succeeds only when:

- the QR code exists in iFound inventory
- the QR code is valid
- the QR code is still `UNASSIGNED`

Registration fails when:

- the QR code does not exist
- the QR code was already assigned to another owner
- required item details are missing

## Managing Registered QR Codes

On the QR Codes page, you can:

- view your registered QR codes
- download QR images
- edit what an item is tagged to
- retire a QR code
- reactivate a retired QR code

## Finder Interaction Flow

When someone scans your QR code:

1. They open the QR landing page.
2. They can send you a message.
3. They can start a call if the feature is enabled.
4. The event is recorded in your account activity.

## Monitoring Activity

Open `Activity` to view:

- scans
- call attempts
- finder messages
- timestamps
- locations if available

## Calls

Owners can join the call flow from the Calls page.

Use this when:

- you are expecting finder contact
- you want to speak with the finder directly from the browser

## Payments and Subscription

Open `Payments` to view:

- subscription status
- revenue and billing metrics
- recorded payment history

## Account Settings

Owners can manage:

- email
- phone number
- linked Google or Apple account
- password setup for social accounts

## Cross-Platform Usage

An owner can:

- register a QR code on web and later manage it in mobile app
- register in mobile app and later manage on web

The same account should reflect the same QR ownership and item data across platforms.

## Common Problems

### QR code rejected during registration

Possible reasons:

- QR was typed incorrectly
- QR code is not part of iFound inventory
- QR code is already assigned

### I signed in but cannot see my QR code

Check:

- you are using the same account
- registration completed successfully
- app and web are using the same backend environment

### Finder contacted me but I do not see it

Check:

- `Activity`
- `Calls`
- your app/browser notifications

## Best Practice

- Register QR codes immediately after receiving the pack
- Use clear item names and categories
- Keep your contact methods current in Settings
